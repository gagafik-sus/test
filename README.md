# 📦 Adubam Package Manager (APM)

**APM (Adubam Package Manager)** is a simple and lightweight package manager for **AdubamOS**.  
It allows you to install, manage, and run applications inside the AdubamOS environment.

---

## 🔗 AdubamOS

👉 https://github.com/gagafik-sus/adubamos

---

## ✨ Features

- Install apps 📥  
- Manage packages ⚙️  
- Lightweight and fast ⚡  
- Designed for AdubamOS ecosystem 🧩  

---

## 🚀 Usage

APM is built into AdubamOS.

Run inside the shell:

```bash
apm
```

---

## ⚠️ Notes

- Works only inside AdubamOS  
- Experimental, may have bugs
